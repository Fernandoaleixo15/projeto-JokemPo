<h1>🎮 Jokenpô - Pedra, Papel ou Tesoura</h1>
<br>
<p>Um clássico jogo de Jokenpô desenvolvido com tecnologias web fundamentais para praticar manipulação de DOM e lógica de programação.</p>
<hr>
<h2>🚀 Tecnologias Utilizadas</h2>
<ul>
  <li>HTML5: Estruturação dos elementos da interface.</li>
   <li>CSS3: Estilização, layout responsivo e transições.</li>
   <li>JavaScript (ES6): Lógica do jogo, geração de jogadas do computador e atualização de placar.</li>
</ul>
<hr>
<h2>🛠️ Funcionalidades</h2>
<ol>
  <li>Escolha entre Pedra, Papel ou Tesoura.</li>
   <li>Oponente (IA) com escolha aleatória.</li>
   <li>Cálculo de resultado em tempo real (Vitória, Derrota ou Empate).</li>
  <li>Placar acumulativo de pontuação.</li>
</ol>
